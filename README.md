# Joseph: Basics Of Streamlit

A simple Streamlit app template with all the basics!

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```

- Displaying Code
